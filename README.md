# Подробности

Чтобы запустить программу, откройте **`Logger CSGORun.exe`**.

Файл с логами **`coef.txt`** будет находиться в той же папке, где и **`Logger CSGORun.exe`**.

Для закрытия программы нажмите **`Ctrl+C`**.
